# cryptologist_by_Borod
Все мы являемся держателями огромного количества ценной информации – логинов, паролей, банковских реквизитов и, естественно, все их запомнить нам тяжело, а хранить в текстовом формате на компьютере – небезопасно. Приложение Cryptologist предназначено для шифровки/расшифровки данных, что обеспечивает спокойствие за всю личную информацию, даже, например, при фишинговых атаках. Ведь для мошенников ваши логины будут лишь бесполезным набором символов, а вы, имея приложение и свой уникальный ключ шифрования, без проблем сможете получить доступ к информации в любой момент.

During my Python studies I created a program - an encryptor of texts, passwords and everything else.
What does user need: 1. run the exe file on your computer, 2. enter text, 3. specify the cipher code, then encrypt / decrypt.
